ComboBox绑定自定义数据源，并设置好显示字段、选择项   

数据源及显示字段，设置好`ItemsSource`和`DisplayMemberPath`   
获得选择项，通过绑定`SelectedItem`或者`SelectedValue`     
不设置`SelectedValuePath`时，SelectedValue和SelectedItem同效
